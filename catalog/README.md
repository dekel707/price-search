# קטלוג לסוחרים

אתר נפרד וקריאה בלבד. הוא מקבל מהמערכת הראשית רק דגם, שם מוצר, צבעים שזוהו ומסמכי PDF.

לא מועברים אליו מלאי, כמויות, מחירים, לקוחות, הזמנות, שריונים או הרשאות כתיבה.

מקור הנתונים החי מוגדר ב־`public/catalog-config.js`. אם המקור אינו זמין, האתר מציג את `catalog-fallback.json` שנוצר בבנייה.
