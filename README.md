# מחירון מהיר

אפליקציית ווב עצמאית לחיפוש מחירון לפי מק״ט, דגם או תיאור מוצר. האפליקציה נטענת עם מחירון פתיחה מתוך קובץ האקסל שסופק, ואפשר לעדכן אותה בכל חודש דרך כפתור `עדכון מאקסל`.

אפשר ליצור קטגוריות ידנית, לשייך קטגוריה לכל מק״ט, ולהוסיף הערה לכל פריט. הקטגוריות וההערות נשמרות בנפרד לפי מק״ט, ולכן טעינת אקסל חדש לא מוחקת אותן.

אם קיימים דפי מוצר ב־`public/img`, האפליקציה יוצרת אוטומטית קובץ `specs.json` בזמן הרצה או בנייה. בחיפוש יופיע כפתור `PDF` ליד כל פריט שיש לו דף מוצר תואם לפי הדגם/מק״ט, והלחיצה פותחת את הקובץ הרלוונטי.

יש גם סל הזמנה: מתוך תוצאת חיפוש לוחצים `הוסף לסל`, בוחרים כמות ומחיר יחידה, אפשר להשתמש במחיר המחירון או במחיר האחרון שנשמר להזמנה קודמת, ואז לשמור הזמנה. הזמנות שמורות ניתנות למחיקה, ומספר וואטסאפ קבוע מייצר הודעת הזמנה קצרה ומוכנה לשליחה.

בתחילת סל חדש, הלחיצה הראשונה על `הוסף לסל` פותחת בחירת לקוח. אפשר לבחור לקוח קיים עם השלמה אוטומטית או להקליד שם לקוח להזמנה הזאת, ואז להמשיך להוסיף מוצרים לאותה הזמנה בלי מעבר מסך.

המסך מחולק ללשוניות: חיפוש, סל, קטגוריות והזמנות. בכל הזמנה אפשר להזין שם לקוח, והוא נשמר בהיסטוריה ונכנס להודעת הוואטסאפ.

בלשונית `לקוחות` אפשר לנהל מאגר לקוחות: להוסיף לקוח, לערוך שם, קוד לקוח וטלפון, לבחור לקוח להזמנה נוכחית, ולראות הזמנות עבר לפי לקוח. הזמנות חדשות נשמרות עם שיוך ללקוח, והזמנות ישנות מזוהות לפי שם לקוח כשאפשר.

בפריסה ל-Vercel הנתונים החשובים נשמרים גם בענן דרך Vercel Blob: מחירון אחרון שהועלה, לקוחות, קטגוריות, הערות, הזמנות שמורות, מחירים אחרונים ומספר הוואטסאפ. לכן פתיחה ממכשיר אחר או העלאת אקסל חדש לא מוחקת את ההגדרות וההיסטוריה.

## הרצה מקומית

```bash
npm install
npm run dev
```

במחשב האפליקציה תיפתח ב־`http://localhost:5173/`. בטלפון שמחובר לאותה רשת Wi-Fi אפשר לפתוח את כתובת ה־Network ש־Vite מדפיס במסוף, למשל `http://10.0.0.17:5173/`.

## עדכון חודשי

פותחים את האפליקציה, לוחצים `עדכון מאקסל`, בוחרים קובץ `.xlsx`, והמחירון נשמר בענן בפריסה של Vercel ובדפדפן המקומי כגיבוי.

## פריסה

```bash
npm run build
```

את תיקיית `dist` אפשר להעלות לכל אחסון סטטי, למשל Vercel, Netlify או GitHub Pages.
